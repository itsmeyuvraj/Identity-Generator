## 🚀 Overview
**Identity Generator** is a lightweight, native macOS Menu Bar application built to eliminate the friction of manual test data creation. Designed with a modern **Liquid Glass** aesthetic, it provides instant access to disposable Indian personas and functional email inboxes directly from your system tray.

> **"Built by a QA, for QAs."** — Optimized for speed, reliability, and zero-distraction workflows.

---

## ✨ Key Features

| Feature | Description |
| :--- | :--- |
| **⚡️ One-Click Persona** | Instantly generates a random Indian Name (First + Last). |
| **🛡️ Robust Email Engine** | Built-in **Email Provider Fallback** to ensure 100% uptime during testing. |
| **📋 Auto-Copy** | Click the name or email to copy it to your clipboard instantly. |
| **🔔 Smart Notifications** | Get a macOS banner when an OTP or verification code arrives. |
| **🧊 Liquid Glass UI** | A high-end, frosted-glass interface that matches the macOS aesthetic. |
| **🔄 Background Polling** | Automatic inbox refresh every 7 seconds for instant code delivery. |

---

## 🛠 Why Use This?
As a **QA Engineer**, manual regression testing often involves creating dozens of "throwaway" accounts. This app solves three major pain points:
1. **Context Switching:** No more opening browser tabs just to get a temp email.
2. **High Reliability:** If the primary mail provider fails, the app automatically switches to a backup to keep your test flow moving.
3. **Data Integrity:** No more "test1@test.com" — use real names like *Arjun Bhatia* for cleaner test data.

## 📦 Installation

To get started with the latest build:

1. **Download:** Go to the [Releases](https://github.com/itsmeyuvraj/Identity-Generator/releases) page and download the `.pkg` installer.
2. **Install:** * **Right-click** the `.pkg` file.
   * Select **Open** (to bypass the macOS Gatekeeper warning).
   * Follow the installation prompts.
3. **Run:** Find the **Identity icon** in your macOS Menu Bar!
